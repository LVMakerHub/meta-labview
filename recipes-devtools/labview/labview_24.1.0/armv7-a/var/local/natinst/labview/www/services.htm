LabVIEW Web Services

The LabVIEW Real-Time engine running on the RT target contains an integrated Web server. In addition to publishing the same type of documents as other Web servers, the built-in Web server on the RT target also can publish running VIs.

You can configure the LabVIEW Web Server settings for an RT Target using the RT Target Properties dialog box. You also can configure the Web server using the Application property node.

Web Server: Configuration

By default, the Web server is disabled. To enable the Web server, right-click the RT target you want to modify in the Project Explorer Window and select Properties to display the RT Target Properties dialog box. Select Web Server: Configuration Options from the list on the left and place a checkmark in the Enable Web Server checkbox. Click OK, then right-click the RT target item again and select Deploy to apply the settings to the target. Using the default settings, you now can connect to the server from a browser by entering http://host.domain.com into the browser's URL field (replace host.domain.com with the Internet address of the RT target).

Web Server: Browser Access

By default, all computers have access to the Web server. To specify which computers on the Internet can access the Web server, right-click the RT target you want to modify in the Project Explorer Window and select Properties to display the RT Target Properties dialog box. Select Web Server: Browser Access from the list on the left. You might want to limit access to only a few specific computers (with entries like host1.domain.com and host2.domain.com) or to all computers within a domain (with an entry like *.domain.com). Click OK, then right-click the RT target and select Deploy to apply the settings to the target.

Web Server: Visible VIs

By default, all VIs running on the RT target can be viewed from any computer listed on the Browser Access page of the RT Target Properties dialog box. To configure which VIs can be viewed using the Web server, right-click on the RT target you want to modify in the Project Explorer Window and select Properties to display the RT Target Properties dialog box. Select Web Server: Visible VIs from the list on the left. When you finish changing the settings, click OK, then right-click the RT target and select Deploy to apply the settings to the target.
